# 🎮 Adivinhe os Números

Projeto em Python desenvolvido para demonstrar o uso de estruturas de repetição (`while`), tratamento de exceções (`try/except`) e lógica de programação básica.

---

## 🧠 Objetivo

O jogo consiste em um quiz simples de matemática com 3 perguntas, onde o usuário deve responder corretamente para acumular pontos.

O projeto foi desenvolvido com foco em prática de lógica e manipulação de entrada de dados.

---

## ⚙️ Funcionalidades

- Loop principal com `while True`
- Sistema de pontuação
- Tratamento de erros com `ValueError`
- 3 perguntas matemáticas fixas
- Feedback de acerto e erro

---

## 🛠️ Tecnologias utilizadas

- Python

---

## ▶️ Como executar

Abaixe o Python em seu computador ou utilize no Visual Studio Code e crie um arquivo com o final py e execute!
